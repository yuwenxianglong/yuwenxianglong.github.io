#### 1. 条件操作符



| 操作       | 格式                     | 范例 | RDBMS中的类似语句 |
| ---------- | ------------------------ | ---- | ----------------- |
| 等于       | {\<key>:\<value>}        |      |                   |
| 小于       | {\<key>:{$lt:\<value>}}  |      |                   |
| 小于或等于 | {\<key>:{$lte:\<value>}} |      |                   |
| 大于       | {\<key>:{$gt:\<value>}}  |      |                   |
| 大于或等于 | {\<key>:{$gte:\<value>}} |      |                   |
| 不等于     | {\<key>:{$ne:\<value>}}  |      |                   |
